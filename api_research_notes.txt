## Key Findings from Swagger Documentation

### /offers endpoint
- GET /offers - Provides a list of available offers
- Added `myReservationId` to response when parameter `showOptions` is set to true (10.07.2024)
- Added `obligatoryExtras` list to response (05.12.2023)
- Added `minYearOfBuild`, `maxYearOfBuild` parameters (27.12.2023)
- Added `kind` parameter in the request (29.05.2020)
- Added `sailingAreaId` as additional parameter in request (10.04.2020)
